== 简介
生产应用(PMS)技术方案,是生产应用完整的实施方法论,包括 *基础环境搭建* 、 *应用API技术手册* 、*系统实施与部署*,其中基础环境包括MYSQL数据库、
Zookeeper、zkui、redis、kafka、RabbitMQ、nginx的安装和配置手册，应用API技术手册涵盖：前端框架详细介绍、后端基础框架服务、
标准中心、设备台帐、设备异动、对外接口、路由网关等API。
